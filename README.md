# 原田良太 ポートフォリオサイト

社会福祉法人 宮共生会 理事長 原田良太の個人ポートフォリオサイトです。

**公開URL**: https://harada-a11y.github.io/portfolio/

## 概要

長崎県佐世保市を拠点に、障害のある方の就労支援・生活支援・医療的ケアから地域活性化まで、「必要な場所がないなら自分たちで作る」という信念で事業を展開しています。

本サイトでは以下を掲載しています：

- ストーリー（生い立ちから理事長就任まで）
- 数字で見る宮共生会（事業年数・事業数・職員数・拠点数）
- 事業の4つの柱（就労支援・農福連携・医療的ケア・地域活性化）
- 講演・研修テーマと主な登壇先
- SINCC（自主企画イベント）
- メディア掲載・動画

## 技術スタック

- HTML5
- CSS3（リセット・フレックスボックス・グリッド・メディアクエリ）
- JavaScript（バニラJS。ライブラリ・フレームワーク不使用）
- GitHub Pages によるホスティング
- Google Fonts（Noto Sans JP）

ビルドステップなし、依存パッケージなしの素のWebサイトです。

## アクセシビリティ対応

**WCAG 2.1 AA 準拠**を目標に設計しています。

- `lang="ja"` による言語指定
- セマンティックHTML（`<header>` / `<main>` / `<nav>` / `<footer>` / `<section>`）
- 適切な見出し階層（h1 → h2 → h3）
- 画像の代替テキスト（alt属性）
- 埋め込み動画のtitle属性
- 文字と背景のコントラスト比 4.5:1 以上
- キーボード操作対応（スキップリンク、Escキーでメニュー閉）
- スクリーンリーダー対応（aria-label / aria-expanded / aria-controls）
- モバイル対応（ハンバーガーメニュー、タップ領域44×44px以上）
